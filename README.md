## Описание проекта
Учебный QA pet-проект по тестированию REST API сервиса https://reqres.in/

Цель проекта — отработка навыков API тестирования:
позитивные и негативные сценарии, проверка статус-кодов и структуры ответов.

## Объем тестирования
- GET запросы
- POST запросы
- Проверка статус-кодов
- Валидация response body
- Негативные сценарии

## Артефакты тестирования
- Тестовые примеры API
- План тестирования
- Баг-репорты

## Инструменты
- Ручное тестирование API (Postman / инструменты браузера)
- REST API
- GitHub
