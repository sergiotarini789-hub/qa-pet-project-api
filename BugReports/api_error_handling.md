ID: API-001

Заголовок:
API возвращает статус 201 при создании пользователя без обязательных полей

Шаги:
1. Отправить POST /api/users
2. Передать пустое тело запроса

Ожидаемый результат:
- Status code 400
- Ошибка валидации

Фактический результат:
- Status code 201
- Пользователь создаётся

Серьезность:
Средняя

Приоритет:
Средняя
